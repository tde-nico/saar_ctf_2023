Specification

Turing Machines on Wikipedia.

Turing machines apply a finite set of states to a tape, which consists of bytes. States have a name and a list of actions, which are evaluated in order. Actions can alter the content of the tape (at the current head position) and move the head left/right. As an additional feature, you can use 16 8-bit registers r0...r9, ra-rf.

Actions consist of:

  • A condition (type: a single character or an expression)
  • A list of actions (type: variable=<expression>)
  • The successor state
  • Head movement (positive numbers: left, negative numbers: right)

Expressions can use the special values h (char under the current head) and r0-rf (registers).

 

Example: implement strlen (and write the result after the string)

  1. State C, actions:
    • condition \x00, no actions, successor W, head 1
    • no condition, action r0=r0+1, successor C, head 1
  2. State W, actions:
    • no condition, action h=r0, successor E, head 0
  3. State E, no actions
